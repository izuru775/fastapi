# fastapi
This application is a base for future fastapi applications.

This will include tools and commands that I've used to create the initial application and some other git commands.

---
## 1. Create project dir

💬 Go to the home directory

cd

💬 Create a directory for all your code projects

mkdir code

💬 Enter into that code directory

cd code

💬 Create a directory for this project

mkdir fastapi

💬 Enter into that project directory

cd fastapi

## 2. Create virtual environment

💬 use uv to create the virtual environment as shown in the following guide

https://fastapi.tiangolo.com/virtual-environments/?h=requirements.txt#upgrade-pip

## 3. Add packages to a requirement file

uv pip freeze > requirements.txt
